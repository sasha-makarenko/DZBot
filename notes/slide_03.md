# Конспект: slide_03

## Резюме
Слайд описывает основные алгебраические свойства сложения векторов и умножения на скаляр, которые формируют основу линейной алгебры.

## Ключевые понятия
- Коммутативное свойство сложения векторов
- Ассоциативное свойство сложения векторов
- Свойство нулевого вектора
- Свойство отрицательного вектора
- Распределительные свойства умножения на скаляр

## Термины
- Коммутативное свойство: $\vec{u} + \vec{v} = \vec{v} + \vec{u}$
- Ассоциативное свойство: $\vec{u} + (\vec{v} + \vec{w}) = (\vec{u} + \vec{v}) + \vec{w}$
- Нулевой вектор: $\vec{v} + \vec{0} = \vec{0} + \vec{v} = \vec{v}$
- Отрицательный вектор: $\vec{v} + (-\vec{v}) = \vec{0}$
- Распределительные свойства: $\lambda(\vec{u} + \vec{v}) = \lambda\vec{u} + \lambda\vec{v}$, $(\lambda_1 + \lambda_2)\vec{v} = \lambda_1\vec{v} + \lambda_2\vec{v}$, $\lambda_1(\lambda_2\vec{v}) = (\lambda_1 * \lambda_2)\vec{v}$

## Пояснение
Давайте разберёмся, что означают эти свойства. Коммутативность говорит, что порядок сложения векторов не важен — результат один и тот же. Ассоциативность — это то, что можно складывать векторы по группам, как угодно, и ответ не изменится. Нулевой вектор — это вектор, который при сложении с любым другим вектором не меняет его. Отрицательный вектор — это такой вектор, что при сложении с исходным получается нулевой вектор. Распределительные свойства показывают, как скалярное умножение распределяется на сумму векторов и как умножение скаляров сочетается с умножением на вектор. Эти свойства делают операции с векторами и скалярами предсказуемыми и удобными для вычислений.
