# Конспект: first

## Резюме
Слайд посвящён основным операциям над множествами, включая универсальное множество, объединение, пересечение, разность, симметрическую разность и разделение на непересекающиеся подмножества.

## Ключевые понятия
- Universal set — универсальное множество.
- Union — объединение множеств.
- Intersection — пересечение множеств.
- Disjoint sets — непересекающиеся множества.
- Partition — разбиение множества.
- Absolute complement — абсолютное дополнение.
- Relative complement — относительное дополнение.
- Symmetric difference — симметрическая разность.

## Термины
- Universal set — множество, из которого берутся все рассматриваемые подмножества в данном контексте.
- Union — операция объединения множеств A и B, задаётся как множество всех элементов, принадлежащих хотя бы одному из множеств A или B.
- Intersection — операция пересечения множеств A и B, задаётся как множество всех элементов, принадлежащих обоим множествам A и B.
- Disjoint sets — множество, состоящее из подмножеств, которые не имеют общих элементов.
- Partition — разбиение множества X на непустые подмножества C, такие что каждый элемент X принадлежит ровно одному подмножеству C.
- Absolute complement — дополнение множества A по отношению к универсальному множеству U, задаётся как множество всех элементов из U, не принадлежащих A.
- Relative complement — дополнение множества A по отношению к множеству B, задаётся как множество всех элементов из B, не принадлежащих A.
- Symmetric difference — операция симметрической разности множеств A и B, задаётся как множество всех элементов, принадлежащих ровно одному из множеств A или B.

## Пояснение
Все операции над множествами определяются через элементы и их принадлежность. Универсальное множество U — это базовое множество, из которого строятся все остальные множества в обсуждаемой ситуации. Объединение A ∪ B включает все элементы, которые есть в A или в B. Пересечение A ∩ B включает только элементы, присутствующие одновременно в обоих множествах. Непересекающиеся множества — это такие, что их пересечение пусто, то есть A ∩ B = ∅. Разбиение множества X на подмножества C — это такое разбиение, при котором каждый элемент X попадает в ровно одно из подмножеств C. Абсолютное дополнение \(\overline{A}\) — это множество всех элементов из универсального множества U, не входящих в A. Относительное дополнение A \ B — это множество всех элементов из B, не входящих в A. Симметрическая разность A Δ B — это множество элементов, принадлежащих ровно одному из множеств A или B. Она может быть выражена как (A ∪ B) \ (A ∩ B), или как A Δ B = ∅ и A Δ A = A, а также (A Δ B) Δ C = (A Δ C) Δ (B Δ C).
